Installation:

  1. Download encrypted files by press of ‘ENCRYPTED FILES’ button.

  2. Go to the FreeMP Discord server by press of ‘GET A PASSWORD’ button.

  3. Open ‘password’ channel at the discord server and read all the information.

    3.1. If it contains a password, go to the step 4.

    3.2. If it does not contain the password, follow the instructions written in the channel.

  4. Run PZFreeMP4156.exe (it’s safe)

  5. Read the license. It is important!

  6. Enter the password.

  7. Choose the ProjectZomboid game folder. It have to be like “C:\Program Files\Steam\steamapps\common\ProjectZomboid”

  8. Go next and install.

Server hosting:

  1. Start your game.

  2. Go to server creation.

  3. Disable DoLuaChecksum at the server settings. It is important!

  4. Configure the server.

  5. Go on and host.

Connection:

  1. Go to the server browser.

  2. Choose a server or enter the server data.

  3. Connect.

Hosting via Steam

If you host the server via Steam, you have to press ESC then INVITE YOUR FRIENDS. If you don’t they won’t be able to join. Skip this note if you host -nosteam server or use non-steam game.

Русский

Установка:

  1. Скачайте зашифрованный установщик, нажав кнопку ‘ENCRYPTED FILES’

  2. Зайдите на FreeMP Discord сервер, нажав кнопку ‘GET A PASSWORD’

  3. Откройте канал ‘password’ на сервере Discord и прочитайте всю информацию там.

    3.1. Если Вы нашли пароль, переходите к пункту 4.

    3.2. Если пароля там не оказалось, следуйте инструкциям по получению пароля в канале.

  4. Запустите PZFreeMP4156.exe (это безопасно)

  5. Прочитайте лицензию. Это важно!

  6. Введите пароль.

  7. Выберите папку с игрой ProjectZomboid. Она может располагаться, к примеру, по этому пути: “C:\Program Files\Steam\steamapps\common\ProjectZomboid”

  8. Нажмите далее и завершите установку.

Запуск своего сервера:

  1. Запустите игру.

  2. Нажмите кнопку “Свой Сервер”.

  3. Отключите параметр DoLuaChecksum в настройках сервера. Это важно!

  4. Настройте сервер так, как считаете нужным.

  5. Нажмите кнопку “запустить”.

Подключение:

  1. Зайдите в браузер серверов.

  2. Выберите сервер или введите его данные в соответствующие поля.

  3. Подключайтесь.

Запуск сервера через Steam

Если Вы хотите играть через Steam, Вам необходимо нажать ESC, а затем ПРИГЛАСИТЬ ДРУЗЕЙ. Если Вы этого не сделаете, Ваши друзья не смогут подключиться. Пропустите эту заметку, если Вы запускаете игру с параметром -nosteam или используете не Steam версию игры.
